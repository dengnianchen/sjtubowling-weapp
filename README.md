## 功能及入口列表

全局功能  
┣━绑定球馆账号（1.0）  
┗━新用户信息录入指引（1.1）

首页  
┣━查看最新员工例会记录  
┣━查看本周班表（1.0）  
┗━查看正在进行的投票列表（1.2）  
　┗━投票、查看投票结果（1.2）  
　　┗━分享投票页（1.2）

全家福  
┣━查看在职/离职员工并可根据多种字段排序（1.0）  
┗━查看员工资料页（1.0） 
　┣━拨打电话（1.3）  
　┗━分享资料页（1.3）  

个人中心  
┣━修改个人信息  
┃ ┣━修改个人资料、网站密码、头像（1.1）  
┃ ┗━修改空余时间（1.0）  
┣━反馈建议（1.0）  
┗━查看版本更新记录（1.0）
